# 虫洞科技 Mod

《逃离鸭科夫》虫洞科技系列道具Mod。

## 功能说明

虫洞科技系列包含以下功能：

- **微型虫洞**：一次性使用，传送回家
- **虫洞回溯**：返回记录的撤离位置
- **虫洞手雷**：随机传送范围内的敌人
- **虫洞徽章**：闪避伤害并触发虫洞传送
- **黑洞手雷**：吸引并伤害范围内的敌人
- **虫洞商店终端**：按F8打开商店购买虫洞科技物品

## 安装方法

1. 将 `WormholeTechMod` 文件夹复制到游戏的 `Duckov_Data/Mods` 目录
2. 启动游戏，在主菜单的 Mods 界面启用该Mod

## 文件结构

```
WormholeTechMod/
├── WormholeTechMod.dll    # 主程序
├── info.ini                # Mod配置
└── preview.png             # 预览图（可选）
```
