## 核心概念速查

### TypeID 分配规范
- **官方物品**: 1-9999
- **Mod物品推荐**: 990000+
- **必须唯一**: 冲突会导致物品无法正常工作

### 物品类型对应组件

| 物品类型 | 核心组件 | 文档 |
|----------|----------|------|
| 消耗品 | UsageBehavior | 03-物品使用与技能.md |
| 被动装备 | Effect + EffectTrigger + EffectAction | 04-效果系统.md |
| 近战武器 | ItemAgent_MeleeWeapon / 自定义Attack | 05-战斗与武器系统.md |
| 投掷物 | SkillBase + Projectile组件 | 03-物品使用与技能.md |
| 枪械 | ItemAgent_Gun | 08-游戏API参考.md |

### 关键生命周期方法

```csharp
// ModBehaviour
Start()      → Mod入口，初始化所有内容
Update()     → 每帧更新，调试快捷键
OnDestroy()  → 清理，取消事件监听

// UsageBehavior
CanBeUsed()  → 检查是否可以使用
OnUse()      → 执行使用逻辑

// SkillBase
OnRelease()  → 技能释放时调用

// EffectTrigger
OnMasterSetTargetItem() → 绑定物品时调用
Trigger()    → 触发效果

// EffectAction
OnTriggeredPositive() → 正面触发时执行
OnTriggeredNegative() → 负面触发时执行
```
