# 微型虫洞 Mod

《逃离鸭科夫》一次性传送道具Mod。

## 功能说明

微型虫洞是一个一次性使用的高科技传送道具：

- **即时撤离**：使用后立即传送回家，无需寻找撤离点
- **一次性使用**：使用后物品会被消耗
- **紧急逃生**：适合危急情况下的快速撤离

## 使用方法

1. 在突袭任务中获得微型虫洞物品
2. 将物品装备到快捷栏
3. 在需要撤离时使用物品
4. 确认使用后会立即传送回家

## 注意事项

- 只能在突袭地图中使用
- 在基地中使用无效
- 使用后物品会被消耗，请谨慎使用

## 安装方法

1. 将 `MicroWormholeMod` 文件夹复制到游戏的 `Duckov_Data/Mods` 目录
2. 启动游戏，在主菜单的 Mods 界面启用该Mod
